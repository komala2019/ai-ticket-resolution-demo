Route registration

Declared first in app.routes.ts, ahead of the guarded path: '' entry, so the router matches it directly: 

{{ '{' }} path: 'ticket-resolution',
  loadChildren: () => import('./home/ticket-resolution/ticket-resolution.module')
    .then(m => m.TicketResolutionModule) }

No canActivate: [AuthGuard] — the module is self-contained and renders without a backend, so it never hits the auth redirect.
