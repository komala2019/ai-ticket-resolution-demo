README

AI Ticket Auto-Resolution

An intelligent, human-in-the-loop support layer that classifies incoming issues by confidence, resolves what it can autonomously, and escalates the rest with full context — so no ticket ever vanishes into a black hole.

Angular feature module Lazy-loaded route Dark-mode aware Standalone demo data

Overview

The module presents two perspectives of one workflow. The Customer view is a conversational assistant that diagnoses and resolves issues in plain language. The CS console view is the agent-facing side: a live approval queue of AI-drafted resolutions, a knowledge base that powers matching, and an analytics dashboard that tracks impact. A floating Tweaks panel lets you drag the confidence thresholds and watch the queue re-route in real time.
